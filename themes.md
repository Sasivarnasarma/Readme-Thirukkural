# Themes

This file shows all available themes and a preview image for each theme using the public preview URL pattern:

```
https://rt.sasivarnasarma.eu.org/api?theme=THEME_KEY
```

Replace `THEME_KEY` with any key from `data/themes.json`.

Notes:
- Previews are SVG images returned by the API.
- You can append other query params to change layout or language, for example: `&type=vertical&lang=english&kural=10&border=false`.
